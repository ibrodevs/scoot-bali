# Scoot Bali Booking App

Expo 54 mobile app based on the Scoot Bali website visual style.

## Stack

- Expo `54.0.33`
- React Native
- Google Fonts: Inter + Sora

## Run

```bash
npm install
npm start
```

Useful commands:

```bash
npm run ios
npm run android
npm run web
```

## Project structure

- `App.js` - app shell and simple screen stack
- `src/screens.js` - app screens
- `src/components.js` - shared UI building blocks
- `src/data.js` - demo content, booking helpers, delivery logic
- `src/theme.js` - colors, fonts, shadows
